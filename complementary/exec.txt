
A/execve : https://fr.manpages.org/execve/2
--------------------------------------------

1/role : execve() est une fonction systeme utilise pour :
- creer de nouveaux processus
- ou pour changer le comportement d'un processus existant sans reutiliser le meme 
espace memoire.
Elle fait partie de la famille des fonctions exec en C, qui sont utilisees pour 
- lancer de nouveaux programmes 
- ou modifier le comportement d'un processus existant.
biblitoheque : <unistd.h>

2/fonction :
retour de type int
3 parametres
1ere lettre -> e = prend en compte les environnements variables.

int execve(const char *filename, char *const argv[], char *const envp[]);
-------------------------------------------------------------------------
const char *filename : 
- Specifie le chemin complet du fichier executable a lancer
ex : /usr/bin/myprogram

char *const argv[] : 
- Tableau de chaines contenant les arguments de ligne de commande
- Chaque chaine est un argument separe par " ".
- termine par un pointeur (NULL)

char *const envp[] :
- Tbaleau de chaines contenant le variables d'environnement
- chaque chaine = 1 variable 
- termine par un pointeur (NULL)

Lexique : 

fonction systeme : interagit directement avec le systeme d'exploitation

variable environnement : sont des paires nom/valeur qui sont heritees par les processus crees par un shell. 
ex : PATH / HOME / USER /shell

c/pipe : http://manpagesfr.free.fr/man/man2/pipe.2.html

1/role : Une pipe en C est un mécanisme de communication entre processus qui permet 
à deux processus liés (généralement un parent et son enfant) d'échanger des données. Voici les principaux points à retenir sur les pipes en C :

Fonctionnement : 

Une pipe est un canal de communication unidirectionnel entre deux processus.
Elle se compose de deux extrémités : une pour lire et une pour écrire.
Les données sont traitées selon le principe FIFO (First In First Out).
La fonction pipe() crée une pipe et renvoie deux descripteurs de fichier :
int pipe(int fds[2]);
